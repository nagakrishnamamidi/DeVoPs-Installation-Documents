Sonatype Nexus Repository Server Installation in Ubuntu Linux Server.

Pre-Requisites:

Java (JDK)
Run the below commands to install Java and Nexus.
Install Java

Login as a root user
sudo su -

#sudo apt update

#sudo apt install openjdk-17-jre -y

Verify Java is Installed
command:- #java -version

Now, you can proceed with installing Sonatype Nexus Repository.



Nexus Repository latest Version Click on https://help.sonatype.com/en/download-archives---repository-manager-3.html

Note: As of release 3.85.0, the Unix archive comes bundled with a platform-specific JDK. This means that the Unix archive cannot be used on a Mac. No need to install Java.

Update the Nexus User in Nexus Configuration:
Edit the /opt/nexus/bin/nexus.rc file and set the user to nexus:

#cd nexus/bin/
#ll
#vim nexus.rc
